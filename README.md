# RMDB: Rust Memory Database

The [Lightning Memory-Mapped Database][LMDB] (LMDB), translated automatically
from C to Rust using [Corrode].

[LMDB]: https://symas.com/lmdb/
[Corrode]: https://github.com/jameysharp/corrode

## Status

Not ready to use
